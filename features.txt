List of all MiLa features

* Closures
* Implicit paramaters
* Optional paramaters
* Variadic paramaters
* Type hints
* Inplace assignment (references)
* Inplace operations
* Objects (kind of, more prototype-ish)
* Common string escapes (as well as hex and octal codes)
* Anything as identifiers
Example: var 'Das variable Name ist ein Stück Scheiße' = 90;
